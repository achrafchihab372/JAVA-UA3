# JAVA-UA3
membres : Achraf Chihab , Yassine Khalil

L'application automatise la gestion des résultats académiques grâce aux fonctionnalités suivantes :

Lecture de données CSV : Importation sécurisée des informations depuis notes.csv (gestion des séparateurs ; et des formats de nombres).

Calcul des moyennes : Traitement automatique des notes pour chaque étudiant.

Algorithme de classement : Tri des étudiants par ordre décroissant en fonction de leur moyenne générale.

Exportation des résultats : Génération automatique d'un nouveau fichier nommé classement.csv contenant le palmarès final.

Architecture SOLID : Utilisation d'interfaces pour séparer la logique métier du stockage des données.
